# Architecture Decision Records

Use este diretorio para registrar decisoes arquiteturais do projeto.

Crie um arquivo por decisao:

```txt
YYYY-MM-DD-titulo-curto.md
```

Modelo:

```md
# Titulo da decisao

## Contexto

Qual problema levou a decisao.

## Decisao

O que foi decidido.

## Consequencias

Impactos positivos, negativos e riscos.

## Alternativas consideradas

Opcoes avaliadas e motivo de descarte.
```
